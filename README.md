# JavaScript Path Planning
This project aims to do the path planning for the map generated by [Intel® RealSense™ SLAM Library](https://software.intel.com/sites/products/realsense/slam/developer_guide.html)

# Functions
**loadMap**<br>
**saveAsPgmMap**<br>

node main.js
